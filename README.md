# Kenzy's Cybersecurity Portfolio

## Editing Your Portfolio
- **Home/Projects/Skills**: Edit `index.html`
- **Resources**: Edit `resources.html` or `resources.yml` for your favorite 3rd party links
- **Blog**: Add new Markdown files to `_posts/` (File format: `YYYY-MM-DD-title.md`)
- **Document Upload**: Uses Uploadcare. Register for your own public key and replace the placeholder key in `docs.html` for production use.

Enjoy your cyber-themed portfolio!
